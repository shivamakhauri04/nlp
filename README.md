# Natural Language Processing

A collection of NLP and recommendation system projects.

## Projects

| Project | Description | Techniques |
|---------|-------------|------------|
| [Quora Smart Compose](projects/quora-smart-compose/) | AI-powered smart compose for suggesting sincere Quora question phrasing | Text Generation, Language Modeling, fast.ai |
| [Medium Article Recommender](projects/medium-article-recommender/) | Content-based recommendation system for Medium articles | Graph Theory, Recommendation Systems |

## Getting Started

Each project is self-contained under `projects/<project-name>/`. Navigate to the project directory and follow its README for setup instructions.

```bash
# Clone the repo
git clone https://github.com/shivamakhauri04/nlp.git
cd nlp

# Navigate to a project
cd projects/quora-smart-compose

# Install dependencies
pip install -r requirements.txt
```
