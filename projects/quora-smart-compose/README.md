# Quora Smart Compose of Sincere Questions

AI-powered smart compose feature that suggests sincere, neutral phrasing for Quora questions. Uses language modeling to recommend words that avoid inflammatory or discriminatory tones.

## Dataset

Quora Insincere Questions Classification dataset from Kaggle:
https://www.kaggle.com/c/quora-insincere-questions-classification/data

## Pre-trained Weights

WikiText-103 pre-trained weights from fast.ai:
http://files.fast.ai/models/wt103/

## Blog Post

https://medium.com/@shivamakhauri04/quora-smart-compose-of-sincere-questions-e3398ce59688

## Usage

Open `notebooks/quora_smart_compose.ipynb` to run the notebook.
