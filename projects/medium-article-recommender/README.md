# Medium Article Recommender

Content-based recommendation system that suggests Medium articles based on a user's current work using graph theory approaches.

## Usage

Open `notebooks/recommendation_system.ipynb` to explore the recommendation system.

## Project Structure

```
medium-article-recommender/
├── src/                # Source code and modules
├── notebooks/          # Jupyter notebooks
├── configs/            # Configuration files
├── scripts/            # Utility scripts
├── tests/              # Test suite
├── experiments/        # Experiment tracking
├── checkpoints/        # Model checkpoints
├── data/               # Datasets
├── assets/             # Static assets
├── docs/               # Documentation
├── requirements.txt    # Python dependencies
└── README.md
```
